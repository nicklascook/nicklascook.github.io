Something To Do

For when you're not quite sure of what to do with today.

This was my first large solo project. Users can filter by a number of options in order to select activities that appeal to them. Additionally, by hovering over any box, a 'Learn More' button appears that performs an 'I'm Feeling Lucky' Google search with the activity name as well as 'learn' & 'reddit' search terms.

I learnt a bunch of new things whilst setting up this project. Including setting up and configuring a LAMP web stack remotely, SSH keys and SFTP file transfer. Frameworks used include Bootstrap and jQuery.

Radio Time Machine

Experience the radio from any year you'd like.

This was my introduction to HTML, CSS and Javascript. I worked in a team of three - planning, designing and executing this idea. The radio plays news, radio interviews and other media from a selected year. However, due to an issue with the API, we were not able to also get music to function correctly, and therefore were unable to implement that.

My tasks included:

  • Designing and building each visual element from scratch in Photoshop.
  • Adding the functionality and animations for the dials.
  • Setting up the HTML document and positioning the elements.
